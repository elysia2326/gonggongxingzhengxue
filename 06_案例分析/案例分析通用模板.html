案例分析通用模板

核心写法：理论框架 + 案例事实对应 + 问题判断 + 对策启示。
考场上不要把案例复述一遍，也不要只背理论。每一段都要出现“理论词 + 材料事实 + 判断”。

1. 五段式通用模板

第一段：定主题、选理论

本案例主要反映了公共行政中的【主题】问题，可以运用【主理论】进行分析，并结合【辅助理论】加以说明。该理论主要关注【理论解决的问题】，其核心在于从【维度1】、【维度2】、【维度3】等方面判断行政行为是否符合公共行政的要求。

可替换句：

  • 如果案例是“政府该不该管”：本案例的关键是政府职能边界问题，应运用 政府职能模式 / 外部效率与治理边界 分析政府、市场和社会的责任划分。
  • 如果案例是“权力如何被约束”：本案例的关键是行政权力运行问题，应运用 依法行政 / 行政监督制度框架 / 行政问责制分析权力边界和责任追究。
  • 如果案例是“政策为什么失败”：本案例的关键是决策与执行问题，应运用 科学民主依法决策 / 行政执行过程 / 行政执行评估 分析。
  • 如果案例是“服务为什么低效”：本案例的关键是服务行政和绩效问题，应运用 服务行政 / 行政效率分析框架 / 绩效管理 分析。

第二段：解释理论框架

【主理论】认为，公共行政活动不能只看表面结果，还要看其权力来源、组织安排、运行过程和公共价值。按照该理论，可以从以下几个方面展开：

第一，【维度1】，即【解释该维度】。
第二，【维度2】，即【解释该维度】。
第三，【维度3】，即【解释该维度】。
必要时还要结合【辅助维度】，判断行政行为是否兼顾效率、公平、法治、责任和公共利益。

第三段：对应案例事实

首先，从【维度1】看，材料中【事实1】表明【判断1】。这说明案例中的行政行为在【公共性/权责/程序/效率/公平】方面存在【问题】。

其次，从【维度2】看，材料中【事实2】反映出【判断2】。这说明问题不只是个别人员态度问题，更与【制度安排/组织结构/决策程序/监督机制】有关。

再次，从【维度3】看，材料中【事实3】进一步说明【判断3】。如果不加以纠正，可能导致【低效、失责、失信、不公平、权力异化、公共利益受损】。

第四段：提出改进对策

针对上述问题，应从以下方面改进：第一，【制度/职能层面】要【明确权责、转变职能、完善规则】；第二，【过程/机制层面】要【规范决策、强化执行、健全沟通协调和监督问责】；第三，【价值/结果层面】要【坚持公共利益、依法行政、提高效率并兼顾公平】。

第五段：总结提升

综上，本案例表明，公共行政不能只追求【速度/控制/部门利益/短期指标】，而应在【公共性、法治、责任、效率、公平、服务】之间取得平衡。只有把【主理论】所强调的【核心要求】落实到具体制度和行政过程中，才能提高政府治理能力和公共服务质量。

2. 题型变体模板

2.1 “请分析案例反映的问题”

本案例反映的问题可以概括为【问题概括】。从【理论】看，主要表现为：

  1. 【表现1】：材料中【事实】说明【判断】。
  2. 【表现2】：材料中【事实】说明【判断】。
  3. 【表现3】：材料中【事实】说明【判断】。

其深层原因在于【权责不清/职能错位/程序缺失/监督不足/绩效导向偏差/沟通协调不畅】。这说明公共行政必须通过【理论要求】来改进行政运行。

2.2 “请运用某理论分析案例”

【理论】的核心内容是【核心概括】。它包括【维度1】、【维度2】、【维度3】。结合材料看：

第一，材料中的【事实1】对应【维度1】，说明【判断】。
第二，材料中的【事实2】对应【维度2】，说明【判断】。
第三，材料中的【事实3】对应【维度3】，说明【判断】。

因此，该案例体现了【理论】所揭示的【核心问题】。改进方向是【对策1】、【对策2】、【对策3】。

2.3 “请提出改进建议”

改进本案例中的公共行政问题，应坚持【主价值】和【理论要求】，从三个层面入手：

第一，明确职能和权责。要解决【越位/缺位/错位/推诿】问题，做到职责清晰、权责一致。
第二，规范过程和程序。要完善【决策、执行、监督、问责、预算、采购、沟通】机制，防止行政行为随意化。
第三，强化公共价值导向。要把【公共利益、服务、公平、法治、效率、责任】作为评价标准，避免只看部门利益或短期指标。

2.4 “请评价政府行为”

对本案例中的政府行为应作辩证评价。一方面，政府行为具有【积极面】，体现了【公共服务/效率提升/宏观调控/资源配置】等要求；另一方面，材料也暴露出【问题面】，如【程序不足/权责不清/监督缺位/公平受损/绩效偏差】。从公共行政学角度看，评价政府行为不能只看动机或结果，还要看其是否符合公共性、合法性、责任性和有效性。

3. 常用理论套写句

理论 开头句 分析维度句 结论句
公共性 本案例首先要从公共性角度判断政府行为是否真正服务公共利益。 可以从公共需求、公共利益、公共服务和公共监督四方面分析。 公共行政不能服务于部门便利或局部利益，而应回到公共利益和公众需求。
服务行政 本案例体现了政府从管制导向向服务导向转型的要求。 可从公民本位、回应需求、公正服务、开放参与和法治德治结合分析。 服务型政府建设的关键是让行政过程围绕公民需求和公共利益展开。
行政问责制 本案例涉及职责履行和责任追究问题，适合用行政问责制分析。 可从问责主体、问责对象、职责义务、过错事实、损害后果和问责程序分析。 问责不是简单处分个人，而是把政府责任制度化、常态化。
官僚制理论 本案例体现了官僚制组织的规范优势及其可能的僵化问题。 可从层级、分工、规则、非人格化、专业化和效率困境分析。 官僚制不能等同于官僚主义，关键是发挥规则优势并克服僵化。
行政权力纵横结构 本案例的关键在于行政权力在上下级和同级部门之间配置不当。 可从纵向授权、横向分工、协调制衡和权责一致分析。 权力配置应做到层级清楚、横向协同、权责一致。
政府职能模式 本案例反映了政府职能边界和政府与市场、社会关系问题。 可从保护型、干预型、引导型以及越位、缺位、错位分析。 政府职能不是越多越好，而要与公共需要、市场失灵和社会能力相匹配。
治理理论 本案例涉及复杂公共事务，不能只依靠单一政府部门解决。 可从政府、市场、社会组织、公民参与、合作网络和责任分担分析。 治理强调多元主体协同，但政府仍要承担引导、协调和监管责任。
行政协调三模式 本案例中的冲突本质上是行政协调不足。 可先判断冲突属于目标利益、思想行为还是政策制度不协调，再分别处理。 协调要针对矛盾来源，不能把所有问题都简单归为态度问题。
科学民主依法决策 本案例暴露出重大行政决策不够科学、民主、依法。 可从信息调研、专家论证、公众参与、风险评估、合法性审查和责任追究分析。 重大决策不能以领导意志替代规范程序。
行政执行过程 本案例的问题不只在政策本身，也在执行过程。 可从准备、实施、总结，以及计划、指挥、制度、协调、监控分析。 政策能否落地，取决于执行过程是否有资源、有协调、有监督、有反馈。
公共预算监督 本案例涉及公共资金使用，应从预算监督角度分析。 可从预算编制、审批、执行、监督和决算分析。 预算是公共财政的法治约束工具，不只是财务安排。
依法行政 本案例的核心是行政权力是否依法取得、依法行使、依法受监督。 可从主体权限法定、程序法定、行政公正和责任行政分析。 依法行政要求行政权力有边界、有程序、有救济、有责任。
行政效率分析框架 本案例中的效率问题不能简单理解为速度慢或成本高。 可从行政资源、行政产出、行政效益、目标正确和公共价值分析。 真正的行政效率是资源节约、产出有效和公共价值正当的统一。
绩效管理 本案例反映了公共部门绩效管理的问题。 可从使命、绩效计划、持续管理、衡量评估和反馈改进分析。 绩效管理应服务持续改进，不能退化为排名、痕迹和数字考核。
效率与公平关系框架 本案例体现了公共行政中效率与公平的张力。 可从效率基础、公平保障、二者悖论和制度平衡分析。 公共行政不能片面追求速度或平均主义，而要实现效率与公平的适度统一。

4. 答题检查清单

  • 是否先判断案例主题，而不是直接背理论。
  • 是否只选 1 个主理论，最多补 1-2 个辅助理论。
  • 是否每个理论维度都对应了材料中的具体事实。
  • 是否写出了问题性质，而不是只复述材料。
  • 是否提出公共行政学意义上的对策，如职能转变、权责一致、依法行政、监督问责、绩效改进、公众参与。
  • 是否回到公共利益、责任、效率、公平、法治和服务等核心价值。

使用提醒

  • 先选理论，再套框架，不要先写材料感想。
  • 每一个理论维度都要对应案例事实。
  • 结论必须回到公共行政学问题，例如治理、组织、责任、效率、公平、公共价值、制度安排等。
  • 具体理论请配合 案例分析理论库 和 案例分析理论选择表 使用。
