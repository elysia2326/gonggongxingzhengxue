第10章 行政生态与行政发展 作为社会系统中的一个子系统，公共行政与外部环境之间存在着密切的互动关系。环境不仅决定着公共行政的性质、目的、形式和策略，而且影响着公共行政的变革与发展。同时，公共行政也对行政环境具有重要的反作用。 当前，人类社会正处于迅猛变革的时代，尤其是后工业社会的来临，使得人类社会的复杂性、不确定性愈加明显。在此，公共行政应不断地进行自我主动的变动和调整，以适应外部环境的变化，做好复杂性与不确定性条件下的危机管理。

10.2.1 行政改革概述

  1. 涵义 → 是指为了适应行政环境的变化和行政系统内部的要求而对公共行政的组织、人员、技术、制度和观念等进行的有意识地创新、发展和调整的过程。
  2. 相关概念 → 行政改革与行政变革 → 行政改革与行政创新 → 行政改革与机构改革 → 行政改革与行政发展 10.2 行政改革

10.2.2 行政改革的动因

  1. 外部动因 → 是指行政环境因素所引起的行政改革的动力。包括经济、政治、文化、技术 等外部因素的变动等。
  2. 内部动因 → 指的是行政系统内部所引起的改革动力。 （1）行政组织的演化 （2）组织分工的细化 （3）行政权力的异化 （4）行政文化的僵化

  1. 涵义 → 行政改革的目标是改革者所确定的行政改革所要取得的结果。
  2. 特性 → 指向性 → 具体性 → 主观性 → 客观性

10.2.4 行政改革的模式

  1. 涵义 → 行政改革的模式，就是在行政改革理论和经验的基础上形成的相对固定的行政改革的方式。
  2. 类型 → 以组织为中心的改革和以技术为中心的改革 → 以外延为主的改革和以内涵为主的改革 → 以理性为典范的改革和以民主为典范的改革 → 基于设计的改革和基于调适的改革

10.3.1 危机管理的含义

  1. 危机 → 是指个人、群体或者组织由于突发事件的出现而可能受到破坏状态，是严重地威胁到其正常的生存与发展的状态。
  2. 危机管理 → 政府或其他社会公共组织通过监测、预警、预控、预防、应急处理、评估、恢复等措施，防止可能发生的危机，处理已经发生的危机，以减少损失，甚至将危险转化为机会，保护公民的人身安全和财产，维护社会和国家安全。 10.3 危机管理

10.3.2 危机管理的模式

  1. 奥古斯丁的六阶段模式 → 危机的避免、危机管理的准备、危机的确认、危机的控制、危机的解决、从危机中获利
  2. 罗伯特·希斯的4R模式 → 缩减阶段、预备阶段、反应阶段、恢复阶段
  3. 米特罗夫和皮尔森的五阶段模式 → 信号侦测阶段、准备及预防阶段、损失控制阶段、恢复阶段、学习阶段
  4. 国内学者的四阶段模式 → 监测、预警阶段；预控、预防阶段；应急处理阶段；评估、恢复阶段
