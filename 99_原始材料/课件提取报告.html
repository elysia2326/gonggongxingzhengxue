2. 处理方式

  1. 解压 ZIP，并过滤 __MACOSX、.DS_Store、._*、~$* 等无效文件。
  2. 将老格式 .ppt 通过 LibreOffice 转换为 .pptx。
  3. 使用 python-pptx 按幻灯片顺序提取文本，生成每章 Markdown。
  4. 将 .pptx 通过 LibreOffice 转换为 PDF，再用 Poppler 将每页幻灯片渲染为 JPEG 图片。
  5. 将每页幻灯片图片插入对应章节 Markdown 的 ## Slide X 小节下方。

说明：原计划使用 convert-to-markdown 技能脚本的 MarkItDown 路线，但当前 Python 环境缺少 markitdown 包，因此改用 python-pptx 做结构化文本提取。LibreOffice 转换时出现过环境提示，但 10 个 .pptx 中间文件均成功生成。

5. 后续使用提醒

  • 章节整理时，课件 Markdown 是主要课件依据。
  • 若课件 Markdown 中某页出现 [No extractable text on this slide]，说明该页文字未能提取，但对应幻灯片图片已经保留在该页小节下方。
  • 如果需要核对动画、备注、隐藏对象或更高分辨率细节，应回看 课件原始文件 中对应 PPT。
  • 题库答案整理时，课件口径优先；如课件 Markdown 提取不完整，应回查原始课件。
